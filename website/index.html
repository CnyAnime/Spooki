Questions & Answers

Frequently Asked Questions

While we are always trying to keep the bot online, sometimes we just have to restart the bot to add breaking changes. You can join our community discord server to be reminded when Spooki goes offline.

If you need any help with the bot, you should join our community discord server where you can ask the developers anything and they will try their best to answer.

If you want to suggest a new feature/report a bug, simply use the /suggest command in our community discord server. The command will ask you if you want to suggest a new feature or report a bug.

We are currently not looking for developers, but the bot is open-sourced on GitHub, meaning you can make a pull request/issue on the GitHub and we will look into it. You will also get the Contributors role in the community discord server for doing this.
