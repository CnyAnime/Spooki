awesome features

Spooki has a lot of features. But we aren't able to list all of them here. So here are the most popular ones.

Utility

We have a lot of utility commands. These commands are very useful and vary from a user-info command to a literal wikipedia command!

Moderation

Moderation commands are useful in almost any server. This is why Spooki is here to help with that. Spooki has a lot of easy-to-use yet advanced moderation commands

Images

Our image module is amazing! You can use it to manipulate images (avatars, emojis etc) or just get pictures of cats & more!
